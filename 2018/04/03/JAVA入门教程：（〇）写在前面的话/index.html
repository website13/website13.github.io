Java入门教程：（〇）写在前面的话

发表于2018-04-03|更新于2022-07-31|Java
|阅读量:

很早之前就想写这么一个东西了，只是觉得自己水平有限，而且没有视频，怕弄不好这个东西。毕竟java学习资料太多了，网上一搜一大把，比如百度“尚学堂视频”，点进去后，各种视频都有。不过一个马士兵的java基础都能录300集，我也是醉了。所以，我想写一个java教程，打算把最实用的java技术写给不知道怎么入门的人。

如果你是一名拥有java基础的人，现在可以把我的页面关了，因为你可能会觉得我罗里吧嗦的讲一些废话。。。我打算把这个入门级别的教程写的啰嗦一点，我知道有些时候一句话就能解释的问题对于初学者来说还是一头雾水。如果有不懂的问题，请直接留言在当篇文章下面，我打算每天都看下留言（逢年过节，周末双休除外）。

我不打算在教程中解释一些数据结构、堆栈、网络编程等等东西，我认为那是另外一门课，我只在这里告诉你什么可以用、为什么不可以这样写，我会告诉你宏观上的原因，不会详细解释底层的东西。很多东西都是先会用，以后工作时间长了，就会幡然醒悟之前不懂的东西了。只希望这是带你走进程序员的启蒙教程，一个快速入门的而非高、大、全的东西。

我打算在每篇文章的最后留一些作业题，把答案放在下一篇博客的末尾，希望各位能认真思考，把题好好的做一做，对自己负责。另外，有过有机会，我会做数据库和java web的入门级别的教程。

为何我这篇的标题是从零开始的？以后你就明白了。

如果有读者发现了错别字，可以留言告诉我，我会及时更正的，谢谢支持。
